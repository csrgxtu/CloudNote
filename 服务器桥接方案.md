### SSH服务器桥接方案
学校有五台高性能服务器，但是处于内网中，如172.16.79.105。我有一台公网服务器:a.b.c.d，想通过任何局域网通过公网服务器间接SSH访问学校内部的服务器。

#### 方案一
这个方案使用ssh建立tunnel
```bash
#　校内服务器
ssh -fCNR 8888:localhost:22 user@a.b.c.d

# 公网服务器
ssh -fCNL *:8889:localhost:8888 localhost

# 测试
ssh -p 8889 user@a.b.c.d
```
